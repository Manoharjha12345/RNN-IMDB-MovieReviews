#  RNN IMDB Movie Reviews Sentiment Analysis

This project is a Deep Learning application that predicts whether an IMDB movie review is **Positive** or **Negative** using a Simple Recurrent Neural Network (Simple RNN).

##  Features

- Predicts movie review sentiment
- Built using TensorFlow/Keras
- Interactive Streamlit web application
- Real-time sentiment prediction
- Uses the IMDB Movie Reviews dataset

##  Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Streamlit
- Jupyter Notebook

##  Project Structure

##  Installation

Clone the repository:

```bash
git clone https://github.com/Manoharjha12345/RNN-IMDB-MovieReviews.git
```

Go to the project directory:

```bash
cd RNN-IMDB-MovieReviews
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

##  Run the Application

```bash
streamlit run main.py
```

##  Model Architecture

- Embedding Layer
- Simple RNN Layer
- Dense Layer
- Sigmoid Activation Function

##  Dataset

The project uses the **IMDB Movie Reviews Dataset** available in TensorFlow/Keras.
